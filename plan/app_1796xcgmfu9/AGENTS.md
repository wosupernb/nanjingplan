# 南京三日游攻略 - 需求拆解文档

## 产品概述

- **产品类型**: 旅游攻略指南网页（展示型单页）
- **场景类型**: <scene_type>info_viz</scene_type>
- **目标用户**: 学生党、预算敏感型旅行者、南京自由行游客
- **核心价值**: 提供一份视觉高级、信息清晰、可直接参考执行的南京三日游省钱攻略
- **界面语言**: 中文
- **主题偏好**: 浅色（暖色调：梧桐绿、城墙灰、秦淮金）
- **导航模式**: 锚点导航（单页滚动）

---

## 内容理解

### 核心主题
一份面向学生党的南京三日游极致省钱攻略指南，涵盖行程规划、预算控制、交通票务、景点预约、避坑建议和装备清单。

### 章节结构
1. **Hero 首屏区** - 标题、核心信息概览，建立第一印象
2. **行程总览** - 4天核心行程节点的时间线概览
3. **逐日详情** - 3天详细行程卡片（时间线+景点+交通+费用）
4. **预算总览** - 费用 breakdown 表格/图表
5. **交通票务** - 去程机票和返程火车信息卡片
6. **景点预约提醒** - 重要预约时间节点
7. **避坑指南** - 精选避坑小贴士
8. **出行装备清单** - 分类展示必备物品

### 重点内容
- **核心结论**: 人均1518-1538元可完成南京三日游，总控1700元
- **关键决策**: 南京博物院需提前7天预约、住宿选夫子庙/三山街区域性价比最高

# UI 设计指南

## 1. 设计推导依据

- **参考意图**: Free Direction —— 无参考材料，从南京城市语义与旅行攻略内容出发自主建立视觉方向
- **核心情绪 / 应用类型**: 学生旅行攻略 / 轻量内容站 —— 理性克制中透出人文温度，像一本精心编排的独立旅行手册而非商业旅游广告
- **独特记忆点**: 梧桐绿作为贯穿全站的时间线锚点色，所有行程节点、预约提醒、费用数字都沿一条"梧桐绿轴线"展开，形成统一的阅读节奏

## 2. Art Direction

- **方向名**: 梧桐纪行
- **Design Style**: Editorial 经典排版 + Warm Natural 自然暖调 —— 大量留白与清晰网格承载密集行程信息，暖灰基底与梧桐绿点缀还原南京街头梧桐成荫的安静质感
- **DNA 参数**: 圆角 subtle（`rounded-md`）/ 阴影 subtle（`shadow-sm`，仅卡片微浮）/ 间距 spacious（`gap-6`~`gap-8`，段落间大量留白）/ 字体方向 衬线标题 + 无衬线正文 / 装饰手法 细线时间轴 + 低饱和色块标签
- **应用类型**: Content —— 长滚动叙事流，信息层级清晰，卡片与时间线交替

## 3. Color System

**色彩关系**: 暖灰基底 + 梧桐绿主色 + 秦淮金点缀。城墙灰作为大面积阅读背景，梧桐绿承担时间线、主行动与品牌识别，秦淮金仅用于费用数字和关键标签的微亮强调。

**配色设计理由**: 城墙灰（暖灰）还原南京明城墙的石材温度，比冷灰更亲和；梧桐绿取自中山陵梧桐大道的实景记忆，作为全站唯一高识别色，承担时间线节点、预约提醒图标、费用高亮；秦淮金饱和度压低至接近哑光铜，仅用于金额数字和少量标签，避免"金色"的廉价感。

**主色推导**: 梧桐绿 `hsl(87 28% 38%)` —— 低饱和深绿，接近夏日梧桐叶的实际观感，不跳脱、不荧光，在暖灰背景上沉稳可读。

**使用比例**: 60% 暖灰中性 / 30% 浅灰绿辅助 / 10% 梧桐绿 primary

| 角色 | CSS 变量 | Tailwind Class | HSL 值 | 设计说明 |
|---|---|---|---|---|
| bg | `--background` | `bg-background` | hsl(36 15% 96%) | 暖灰基底，模拟纸张与城墙石材质感 |
| card | `--card` | `bg-card` | hsl(36 20% 99%) | 微暖白卡片，与 bg 拉开微妙层次 |
| text | `--foreground` | `text-foreground` | hsl(30 8% 18%) | 深棕灰正文，比纯黑柔和 |
| textMuted | `--muted-foreground` | `text-muted-foreground` | hsl(30 5% 48%) | 辅助说明、交通信息、时间标注 |
| primary | `--primary` | `bg-primary` / `text-primary` | hsl(87 28% 38%) | 梧桐绿：时间线节点、预约标签、主按钮 |
| primaryForeground | `--primary-foreground` | `text-primary-foreground` | hsl(36 20% 99%) | primary 上的白色文字 |
| accent | `--accent` | `bg-accent` | hsl(87 20% 92%) | 极浅梧桐绿底：hover、选中、标签浅底 |
| accentForeground | `--accent-foreground` | `text-accent-foreground` | hsl(87 25% 32%) | accent 上的深绿文字 |
| border | `--border` | `border-border` | hsl(30 8% 85%) | 暖灰边界，卡片与时间线分隔 |

**语义色提示**:
- **秦淮金（费用强调）**: `hsl(38 45% 48%)` —— 仅用于金额数字和预算表关键行，bg 态 `hsl(38 45% 94%)`，不使用 border 态；饱和度与 primary 对齐，不刺眼
- **预约提醒（警告）**: `hsl(28 65% 52%)` —— 预约截止标签，bg `hsl(28 65% 94%)`，border `hsl(28 55% 78%)`，text `hsl(28 60% 35%)`；暖橙调与梧桐绿同属自然色系
- **成功（免费标识）**: `hsl(150 25% 38%)` —— "免费"标签，bg `hsl(150 20% 92%)`，border `hsl(150 18% 78%)`，text `hsl(150 28% 30%)`

## 4. 字体与节奏

- **font-display**: Noto Serif SC —— 中文衬线用于 Hero 标题和日期标题，传递旅行手册的编辑感与人文温度
- **font-body**: Noto Sans SC —— 无衬线正文保证行程信息、费用表格、预约提醒的高密度可读性
- **字号**: H1 text-5xl ~ text-6xl（Hero 大标题）；H2 text-2xl ~ text-3xl（章节标题）；body text-base；muted text-sm
- **圆角**: subtle —— `rounded-md`，卡片、按钮、标签统一微圆角，保持克制不甜腻

## 5. 全局布局契约

- **Reference Layout Use**: 按需求结构推导 —— Hero → 行程总览时间线 → 逐日详情卡片 → 预算表 → 交通票务卡片 → 预约提醒 → 避坑指南 → 装备清单
- **Page / Section Order**: 与需求文档 8 个区块顺序 1:1 对齐，长滚动单页
- **Standard Content Zone**: `max-w-3xl mx-auto` —— 旅行手册式窄栏阅读，单行 65-75 字符
- **Shell / Frame Alignment**: 内容容器独立滚动，顶部固定极简导航（仅站点标题 + 目录锚点）
- **Padding & Rhythm**: `px-4 md:px-6 lg:px-8 py-16 md:py-24`，章节间距 `gap-16 md:gap-24`
- **Full-bleed Zones**: Hero 区域背景可 `w-full`，内部文字仍受 `max-w-3xl` 约束；逐日详情卡片可略微突破内容区形成视觉节奏
- **Local Narrowing**: 预算表格、装备清单在 `max-w-2xl` 内居中
- **Overflow Strategy**: 预算表在移动端使用 `overflow-x-auto`；时间线在窄屏转为垂直列表
- **Flexibility Boundary**: 允许移动端调整卡片内边距和章节间距；不允许改变 `max-w-3xl`、梧桐绿色值、圆角系统

## 6. 视觉与动效

- **装饰**: 细线时间轴 + 低饱和色块标签 —— 时间线使用 `border-l-2 border-primary` 细线贯穿每日行程，节点为小圆点；预约提醒使用 accent 浅底标签
- **阴影/边界**: 轻 —— 卡片 `shadow-sm` 微浮，其余以 border 分隔为主
- **动效**: 精致 —— 时间线节点 hover 放大至 1.2x + 颜色加深；卡片 hover 轻微上浮 4px + 阴影加深；页面滚动触发章节淡入；费用数字 hover 显示秦淮金色

## 7. 组件原则

- 按钮、标签、卡片必须有 Default / Hover / Active / Focus / Disabled 状态
- Primary 按钮（梧桐绿底白字）仅用于"查看详情""复制预约链接"等主行动；次要操作用 Outline 或 Ghost
- 时间线节点：默认梧桐绿小圆点，hover 放大 + 深绿，当前阅读章节节点填充实心
- 预约提醒标签：警告橙浅底 + 深橙文字 + 左侧小图标；免费标签用成功绿浅底
- 费用数字：默认 text 色，hover 切换为秦淮金 `hsl(38 45% 48%)`

## 9. Anti-patterns

- **Split personality**: 不同章节切换卡片圆角或阴影强度；全站统一 `rounded-md` + `shadow-sm`
- **Phantom tokens**: 编造不存在的 CSS 变量；仅使用已定义 9 角色色板
- **Default SaaS drift**: 回退到蓝色链接、紫色渐变按钮、通用卡片堆叠；所有交互色锚定梧桐绿
- **Invisible interaction**: 时间线节点、卡片、标签 hover 做了但 focus-visible 丢失；所有可交互元素必须有键盘可见焦点环（梧桐绿 `ring-2 ring-primary`）
- **Mono-hue tyranny**: 梧桐绿同时用于按钮、链接、图标、边框、时间线、标签；按 60-30-10 收回，链接和边框用暖灰，图标用 textMuted，仅时间线节点和主按钮用 primary
- **Status color drift**: 预约警告橙饱和度远高于梧桐绿；警告橙饱和度控制在 65%，与 primary 的 28% 饱和度保持同一年代感
- **Image noise**: 景点配图风格不统一，一张高饱和一张黑白一张冷调；所有图片统一低饱和暖调处理
