# 风景图片审美评分

## 功能
对风景图片进行专业审美评分，基于五个核心维度。

## 评分维度（每项20分，总分100分）

### 1. 构图（20分）
- 三分法则运用
- 引导线效果
- 对称性与平衡
- 前景/背景层次
- 主体位置

### 2. 色彩（20分）
- 色彩和谐度
- 饱和度适中
- 对比度
- 色彩丰富性
- 色调统一性

### 3. 光线（20分）
- 光线方向
- 光影效果
- 曝光准确性
- 特殊光线（丁达尔、黄金时刻等）
- 明暗过渡

### 4. 清晰度（20分）
- 分辨率（2K+优先）
- 对焦准确性
- 细节表现
- 锐度
- 无模糊/噪点

### 5. 视觉吸引力（20分）
- 主题突出性
- 情感表达
- 独特性
- 整体美感
- 记忆点

## 评分标准
- 90-100分：卓越，专业级作品
- 80-89分：优秀，高质量图片
- 70-79分：良好，值得保留
- 60-69分：一般，有明显缺陷
- 60分以下：不合格

## 使用方法
1. 读取图片文件
2. 分析五个维度
3. 输出各维度得分和总分
4. 给出简短评价

## 输出格式
```
图片：[文件名]
构图：XX/20
色彩：XX/20
光线：XX/20
清晰度：XX/20
视觉吸引力：XX/20
总分：XX/100
评价：[简短描述]
```
